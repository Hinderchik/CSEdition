## Что это?

CS Edition — мод для Minecraft 1.20.1 (Forge), который добавляет режим Counter-Strike:
покупка оружия, раунды, команды T и CT, зоны закупки, спавны.

Работает на базе **TaCZ** (Timeless and Classics Zero) — все пушки и патроны оттуда.

---

## Команды

### Управление матчем

| Команда | Что делает |
|---------|------------|
| `/cs help` | Список всех команд |
| `/cs start [mapId]` | Начать матч на карте (или текущей если не указана) |
| `/cs stop` | Остановить матч, всех в лобби |
| `/cs status` | Текущая фаза, карта, режим, таймер, кол-во игроков |
| `/cs maps` | Список всех карт |
| `/cs money <amount>` | Добавить себе денег |

### Режимы

| Команда | Что делает |
|---------|------------|
| `/cs mode <modeId>` | Установить режим (classic, deathmatch, gungame, pistol_only) |
| `/cs modes` | Список всех режимов |
| `/cs setmapmode <mapId> <modeId>` | Привязать режим к карте |

### Выдача предметов (только для админов)

| Команда | Что делает |
|---------|------------|
| `/cs give <player> <gunId>` | Выдать оружие игроку |
| `/cs giveammo <player> <gunId> [amount]` | Выдать патроны (по умолчанию 3 магазина) |

Примеры:
```
/cs give XunKal1 glock_17
/cs giveammo XunKal1 ak47 120
```

### Управление картами (нужен op-уровень)

| Команда | Что делает |
|---------|------------|
| `/cs setlobby` | Установить лобби = твоя позиция |
| `/cs addmap <id> [name]` | Добавить/обновить карту |
| `/cs delmap <id>` | Удалить карту |
| `/cs setspawn <mapId> <T\|CT>` | Добавить спавн для команды |
| `/cs clearspawns <mapId> <T\|CT>` | Очистить спавны команды |
| `/cs setbuyzone <mapId> <T\|CT>` | Установить зону закупки (2 угла) |
| `/cs reload` | Перечитать конфиги с диска |

### Тестовые команды (op-only)

| Команда | Что делает |
|---------|------------|
| `/cs test phase <LOBBY\|BUY_TIME\|FIGHTING\|ROUND_END>` | Переключить фазу |
| `/cs test team <T\|CT>` | Установить свою команду |
| `/cs test spawn` | Телепорт на спавн |
| `/cs test buy <gunId>` | Попробовать купить оружие |
| `/cs test give <gunId>` | Выдать себе оружие |
| `/cs test setbuyzone <T\|CT>` | Зона закупки вокруг тебя (8 блоков) |

### Настройки

| Команда | Что делает |
|---------|------------|
| `/cs config slots <n>` | Макс. слотов инвентаря (1-36) |
| `/cs config kills <n>` | Киллов для победы (deathmatch) |
| `/cs config rounds <n>` | Раундов для победы в катке (MR mode) |
| `/cs config clearinv <true\|false>` | Очищать инвентарь после матча |
| `/cs config kept add <itemId>` | Добавить предмет в "kept" (не пропадает) |
| `/cs config kept remove <itemId>` | Убрать из kept |
| `/cs config kept list` | Показать kept items |
| `/cs config show` | Показать все настройки |

## Настройка карты

### Шаг 1: Создай мир

Создай обычный мир в Minecraft. Мод не требует специальной генерации.

### Шаг 2: Построй арену

Построй карту как тебе нравится. Например, два здания друг напротив друга с проходом между ними.

### Шаг 3: Установи лобби

Встань туда, где игроки будут спавниться между раундами:
```
/cs setlobby
```

### Шаг 4: Добавь карту

```
/cs addmap de_dust2 Dust 2
```

### Шаг 5: Установи спавны для T

Встань на место спавна T, выполни:
```
/cs setspawn de_dust2 T
```

Перейди на другой спавн T, выполни ещё раз. И так далее — можно добавить несколько спавнов.

### Шаг 6: Установи спавны для CT

Аналогично:
```
/cs setspawn de_dust2 CT
```

### Шаг 7: Установи зону закупки

Встань в один угол зоны закупки T, выполни:
```
/cs setbuyzone de_dust2 T
```

Перейди в противоположный угол, выполни ещё раз — зона готова.

Аналогично для CT:
```
/cs setbuyzone de_dust2 CT
```

### Шаг 8: Проверь

```
/cs status
/cs maps
```

### Шаг 9: Запусти матч

Минимум 2 игрока должны быть онлайн:
```
/cs start de_dust2
```

## Игровой процесс

### Фазы матча

1. **LOBBY** — ожидание игроков. Урон отключён.
2. **BUY_TIME** (15 сек) — можно купить оружие в зоне закупки. Нажми **B** для меню покупки.
3. **FIGHTING** (2 мин) — раунд идёт. Побеждает команда которая убьёт всех врагов или не даст бомбе взорваться за время.
4. **ROUND_END** (5 сек) — показ результата раунда, затем новый раунд.

### Победа в матче

Команда побеждает когда набирает **`roundsToWin`** раундов (по умолчанию 8 в Classic).
Счёт отображается в HUD: `T 5 : 3 CT / 8`

Настройка: `/cs config rounds <n>` (1-30)

### Бинды клавиш

- **B** — открыть меню закупа (в фазе BUY_TIME)
- **M** — открыть выбор режима (в лобби)
- **G** — выбросить оружие
- **F** — осмотреть оружие (TaCZ)
- **Z** — купить последнее оружие
- **X** — купить основное оружие (самый дорогой автомат)
- **C** — купить пистолет
- **4** — купить снаряжение
- **N** — переключить ночное зрение

## Конфигурация

### Файлы конфигов

- `config/csedition-weapons.json` — цены и параметры оружия
- `<world>/data/csedition/csconfig.json` — настройки матча (создаётся автоматически)
- `<world>/data/csedition/maps.json` — карты (создаётся автоматически)

### Формат weapons.json

```json
{
  "weapons": [
    {
      "id": "tacz:glock_17",
      "displayName": "Glock 17",
      "magazineSize": 17,
      "fireMode": "SEMI",
      "price": 200,
      "category": "pistol"
    },
    {
      "id": "tacz:ak47",
      "displayName": "AK-47",
      "magazineSize": 30,
      "fireMode": "AUTO",
      "price": 2700,
      "category": "rifle",
      "attachments": {
        "AttachmentMUZZLE": "tacz:silencer"
      }
    }
  ]
}
```

Категории: `pistol`, `smg`, `rifle`, `sniper`, `heavy`, `utility`

## HUD (кастомный)

- **HP/AP** — слева снизу (красная/синяя полоски)
- **Деньги** — справа сверху
- **Таймер фазы** — сверху по центру (ЗАКУП / БОЙ / КОНЕЦ РАУНДА)
- **Счёт** — под таймером: `T 5 : 3 CT / 8`
- **Хотбар** — справа снизу, вертикальный, 3 слота

## Частые проблемы

**Мод не загружается:** Проверь что установлен TaCZ и Forge 47.x

**Оружие не появляется:** Проверь ID в `/cs list guns`. Должен быть `tacz:glock_17` или просто `glock_17`.

**Зона закупки не работает:** Должна быть установлена через `/cs setbuyzone` с двумя точками (мин и макс углы).

**Броня не выдается:** Используется Netherite с Protection IV. Слот автоматически надевается.

**HUD не виден:** Проверь что фаза не LOBBY. HUD показывается только в BUY_TIME, FIGHTING, ROUND_END.

- **Forge:** 47.x
